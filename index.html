Snake Game

O maior objetivo deste projeto era criar uma inteligência artificial por mapa de calor que pudesse jogar o clássico jogo da cobra. Nesse projeto é possível também o usuário jogar utilizando as teclas do teclado. Para ativar a inteligência artificial, basta apertar o botão IA e se quiser ver o mapa de calor gerado, basta apertar a luzinha que acende no lado do botão principal (esta função foi feita para debug).

Minesweeper

Assim como no jogo da cobra, o maior objetivo deste projeto foi criar uma inteligência artificial capaz de jogar o clássico jogo Campo Minado. Devido a dificuldade do jogo ela não é capaz de ganhar todos, pois em alguns momentos deve fazer alguma escolha aleatória. O usuário pode jogar também para comparar com a inteligência artificial. Para a inteligência artificial fazer uma jogada, basta apertar Espaço e para voltar uma jogada basta apertar Backspace, podendo voltar até 50 jogadas. No canto esquerdo superior é apresentado o tempo de jogo e no canto direito superior a quantidade de minas restantes.
